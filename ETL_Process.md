
# ETL Process

## ETL Process - Simple Explanation

**ETL (Extract, Transform, Load)**

### Extract
Taking data from where it sits (database, files, websites)
Just collecting the raw data, no changes yet

### Transform
Cleaning and organizing the data to make it usable
Removing errors, duplicates, and arranging it properly

### Load
Moving the cleaned data to its final destination (data warehouse, new database)
Making sure it arrives safely and completely
